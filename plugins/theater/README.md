# Theater

This plugin is usefull if many speakers present the talk.

In the speaker notes, you can use brackets to specify who speaks.

```
/*

{Juliet} Ay me!

{Romeo} She speaks.

{Juliet} O Romeo, Romeo! wherefore art thou Romeo?

*/
```

Then you can add css to stylify your dialog

```css
.Romeo {
    color: red;
}

.Juliet {
    color: blue;
}
```

A `<small>` html element is used to display names, then you can style it too.
